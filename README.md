# Analyse Offensive de PCB

Ceci est un carnet d'expérimentations. Par manque de temps à y consacrer, il restera inachevé. Actuellement, il comporte probablement des erreurs, des approximations, des placeholders et des tournures de phrases incorrectes. Les sujets traités le sont pour certains partiellement, pour d'autres completement. Il est à prendre en l'état avec ses qualités et ses défauts.

J'ai également pu constater après relecture que la bibliographie est incomplète. Je m'excuse d'avance pour les références qui n'ont pas été citées.
